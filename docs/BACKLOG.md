- [~] **2차 조사 진행 중 (2026-07-15)**: 미검증 21종(기존 13 + 신규 8: 가자미·붉바리·능성어·자바리·전복·시마아지·어름돔·점성어) 전수 재검증 + NIFS/공공데이터포털 API·KOGL 확인 + 양식·수입종 '연중' 표기 판단
- [ ] 출처 후보 조사 (아래 후보들의 실제 제공 범위·이용 조건 확인 필요):
